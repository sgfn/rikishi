# rikishi

rikishi -- a **fair** app for moderating sumo competitions

## Getting started

### Requirements

* Java 17 compiler and runtime, e.g. [OpenJDK 17](https://openjdk.org/projects/jdk/17/)
* NodeJS >=20

### Install and run

```bash
./gradlew build
./gradlew run
```

### Test

```bash
./gradlew test
```
